UI-SDK

The Alteryx UI-SDK is a React-based software development kit that lets you build custom applications to use within the Alteryx platform. The UI-SDK provides many product-specific messaging systems. You can use these across the Alteryx platform, as well as the entire core segment of the Alteryx Eclipse Components library. The UI-SDK consists of three components. The React Comms, the Eclipse Component Library, and all associated walk throughs/help guides. Check out the below links to get to each documentation site.
